Mis Servicios

Diseño Web

El diseño web es el proceso de creación de sitios web. Abarca varios aspectos diferentes, incluido el diseño de la página web , la producción de contenido y el diseño gráfico. Si bien los términos diseño web y desarrollo web a menudo se usan indistintamente, el diseño web es técnicamente un subconjunto de la categoría más amplia de desarrollo web.

Aplicaciones Web

Una aplicación web es una aplicación o herramienta informática accesible desde cualquier navegador, bien sea a través de internet (lo habitual) o bien a través de una red local. A través de el navegador se puede acceder a toda la funcionalidad y tener cualquiera de las soluciones enumeradas en el punto anterior.

E-Commerce

En un término amplio el e-commerce hace referencia a todas las transacciones comerciales que operan a través de internet. Sin embargo, regularmente el término se utiliza para describir la comercialización y transacción de productos físicos en internet, es decir una tienda en línea.
